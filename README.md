# CoinGecko API Client for Node.js

## • Quick Start Example

```javascript
//1. Import 
let Geko = require("./lib/geko");

//2. Initiate the CoinGecko API Client
const CoinGeckoClient = new Geko();

//3. Make calls
var func = async() => {
  let data = await CoinGeckoClient.ping();
};
```
